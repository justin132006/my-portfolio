JavaScript Syntax


JavaScript syntax is the set of rules, how JavaScript programs are constructed:

// How to create variables:
var x;
let y;
// How to use variables: x = 5;
y = 6;
let z = x + y;



JavaScript Values


The JavaScript syntax defines two types of values:

Fixed values
Variable values

Fixed values are called Literals.
Variable values are called Variables.


JavaScript Variables


In a programming language, variables are used to store data values.
JavaScript uses the keywords var, let and const to declare variables.
An equal sign is used to assign values to variables.
In this example, x is defined as a variable. Then, x is assigned (given) the value 6:

let x;
x = 6;


JavaScript Operators


JavaScript uses arithmetic operators ( + - * / ) to compute values:


(5 + 6) * 10


JavaScript uses an assignment operator ( = ) to assign values to variables:


let x, y;
x = 5;
y = 6;


JavaScript Expressions


An expression is a combination of values, variables, and operators, which computes to a value.
The computation is called an evaluation.
For example, 5 * 10 evaluates to 50:


5 * 10

Expressions can also contain variable values:

x * 10

The values can be of various types, such as numbers and strings.
For example, "John" + " " + "Doe", evaluates to "John Doe":

"John" + " " + "Doe"

JavaScript Variables


JavaScript is a versatile and widely used programming language that allows you to store and manipulate data using variables. Variables are like containers for data that can hold various types of information, such as numbers, text, and more. In this tutorial, we will cover the fundamentals of JavaScript variables, including how to declare, initialize, and use them effectively


1. Introduction to Variables


In JavaScript, a variable is a named container that stores data. You can think of variables as labels for data, making it easier to work with information in your code. Variables can hold various types of data, from numbers and strings to objects and functions.


2. Declaring Variables


To declare a variable in JavaScript, you use one of three keywords: `var`, `let`, or `const`. Here's how you declare a variable:


JavaScript

var myVariable; // Declaring a variable using 'var'

let anotherVariable; // Declaring a variable using 'let'

conts constantVariable = 42; // Declaring and initializing a constant variable using 'const'


Initializing Variables


After declaring a variable, you can initialize it with a value. Initialization is the process of assigning a value to a variable. Here's how to do it:


javascript

var age; // Declaring a variable
age = 25; // Initializing the variable with a value

let name; // Declaring a variable
name = "John"; // Initializing the variable with a string value

const pi = 3.14; // Declaring and initializing a constant variable
```
You can also declare and initialize a variable in a single step:

```javascript

var count = 10; // Declaring and initializing a variable in one step
```
